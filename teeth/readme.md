a Python3 script to generate bicycle sprocket and chainring shapes in SVG  
<br/><br/>

Usage:

```bash
python3 genTeeth.py <teeth> <tooth angle> <tooth height> <magic number>
```
<br/>

Good chainring teeth:

```bash
python3 genTeeth.py 45 24 4 1.5
```
